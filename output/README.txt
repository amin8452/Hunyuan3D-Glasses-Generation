Ce dossier contient les modèles 3D de lunettes générés par l'API Hunyuan3D.

Les fichiers générés incluent:
- Fichiers GLB: Modèles 3D de lunettes
- Fichiers JSON: Métadonnées associées aux modèles

Ces fichiers sont générés automatiquement lorsque vous exécutez la commande:

python generate_glasses.py --api
